**wolfir: AI that secures your cloud — and secures itself.**

Every company runs AI on AWS. wolfir is the first security platform that secures **both** your cloud infrastructure **and** your AI pipeline — and uses AI to do it. When you use AI to defend your cloud, who defends the AI? wolfir does both, and monitors itself using MITRE ATLAS while it runs.

**Two layers, one platform:**
- **Cloud Security** — Incident response: CloudTrail → detect, investigate, classify, remediate, document. Attack path diagrams, compliance mapping (CIS, NIST, SOC 2, PCI-DSS), cost impact, one-click remediation with CloudTrail proof.
- **AI Security** — MITRE ATLAS (6 techniques), OWASP LLM Top 10, Shadow AI detection, AI Compliance (EU AI Act, NIST AI RMF). "Who protects the AI?" — we watch ourselves.

**Built with 5 Amazon Nova models + Nova Act + Nova Multimodal Embeddings.** Nova Pro for visual architecture (STRIDE), Nova 2 Lite for temporal reasoning and remediation, Nova Micro for risk classification, Nova 2 Sonic for voice (Aria), Nova Canvas for report art, Nova Act for browser automation (AWS Console, JIRA), Nova Embeddings for "find similar incidents."

**Key differentiators:**
- **Secures itself** — MITRE ATLAS self-monitoring. Prompt injection, API abuse, data exfiltration. We watch ourselves.
- **Cross-Incident Memory** — DynamoDB-backed correlation. Run two demos — the second says "78% probability same attacker."
- **Autonomous Remediation with Proof** — Executes AWS API calls. CloudTrail confirmation, one-click rollback.
- **Agentic Pivot** — When timeline confidence is low, the agent runs CloudTrail anomaly scan before proceeding.

**Tech:** Strands Agents SDK, 6 AWS MCP servers (CloudTrail, IAM, CloudWatch, Security Hub, Nova Canvas, AI Security), 23 MCP tools, FastAPI, React, Vite.
